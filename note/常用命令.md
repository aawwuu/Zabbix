### 查看Zabbix 日志

```
cat /var/log/zabbix/zabbix_server.log 
```

### 查看Zabbix 配置

```
vim /etc/zabbix/zabbix_server.conf
```

### 开启10050端口

```
开启端口
firewall-cmd --zone=public --permanent --add-port=10050/tcp
重载服务
firewall-cmd --reload
检测端口是否开启
firewall-cmd --query-port=10050/tcp
```
